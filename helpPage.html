Survival Guide

  • Create an account

    • Instructions

    1. New users can sign up by clicking on the Create New Account link on the Login page. The image is shown below.
    2. Fill out the required fields and click "Sign up".

  • Change Profile Picture

    • Instructions

    1. On the vertical navigation bar present on the left-hand side of the Homepage, you will find 'Select image to upload:' option.
    2. Please click on the Choose file option right below it.
    3. Please choose a *.jpg or *.png file.
    4. Click on upload image after having chosen a file.

  • Sign-out

    • Instructions

    1. Users can logout by clicking on username.

  • Create a new channel

    • Instructions

    1. Users can create a new channel by clicking on the "+" icon next to Channels on the left side in order to create a new channel
    2. Provide a name for your channel, purpose for the channel and add the users you'd like to have on your channel.
    3. Users have options to create either a Public channel in which any user could join or a private channel which requires an invite from the owner of the Channel.
    4. A user will have to remain logged-in for creating a channel.

Messages

  • User search Profile

    • Instructions

    1. Users can look up other users in search bar and view their profile, located on the top right corner of the webpage.

  • Post

    • Instructions

    1. Users can post messages on channels by clicking on '+' icon for images, code snippets, URL images and user can post message in the text field by pressing "Enter" button.
    2. Users can also post images, code snippets and important links.
    3. Users can post images of the formats " *.png, *.jpeg/jpg, *.gif, *.tiff/tif ".
    4. Please note that empty messages cannot be posted.
    5. Please refrain from using profane or vulgar language.

  • Reactions

    • Instructions

    1. Users can react to already existing posts.
    2. Users can either like or dislike a post.
    3. Users can also reply to the existing posts. Replies could be in the form of a message or a code.
